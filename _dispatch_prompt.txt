**Problem 1: External jobs waste 30s trying to click invisible button**

In `src/agent/linkedin_flow.py`, after `classifier.click_apply_button()` for EXTERNAL pages:
- The popup is already captured by TabManager
- But flow enters FormProcessor which extracts DOM from LinkedIn page
- Claude says "click Apply button" 
- Runner tries clicking the now-invisible button â†’ 30s timeout
- Eventually popup is handled, but we wasted 30s

**Fix for linkedin_flow.py:**

After `classifier.click_apply_button()`, if `page_type == PageType.EXTERNAL_LINK`:
1. Wait 1-2 seconds for popup
2. Check `self._tabs.get_captured_popup_url()`
3. If popup URL exists, navigate to it immediately
4. THEN enter FormProcessor

```python
if not classifier.click_apply_button():
    return ApplicationResult(...)

self._page.wait(1500)

# For external jobs, immediately handle popup before FormProcessor
if page_type == PageType.EXTERNAL_LINK:
    popup_url = self._tabs.get_captured_popup_url()
    if popup_url and popup_url != "about:blank":
        logger.info(f"External job: navigating to popup {popup_url}")
        self._page.goto(popup_url)
        self._page.wait(2000)
        self._tabs.close_extras(keep=1)

# Now create FormProcessor with updated page state
dom_service = DomService(self._page)
```

**Problem 2: Dice modal blocks all clicks**

On Dice, a React modal overlay blocks pointer events:
```html
<div data-rac="" class="fixed left-0 top-0 z-modal ... bg-black bg-opacity-50 ...">
```

**Fix for page_classifier.py `_dismiss_overlays()`:**

Add Dice modal selector to the dismiss logic:
```javascript
// Add to the evaluate script:
document.querySelectorAll('[class*="z-modal"][class*="bg-opacity"], [class*="bg-black"][class*="bg-opacity-50"]').forEach(el => {
    if (el.classList.contains('fixed') && el.offsetParent) el.remove();
});
```

**Problem 3: Upload action targets label instead of input**

In `src/executor/runner.py`, the upload action fails when Claude targets a `<label>` instead of the actual `<input type="file">`.

**Fix for runner.py `_execute_action()` UploadAction case:**

When the locator resolves to a label, find and use the associated input:
```python
case UploadAction():
    selector = self._dom.get_selector(action.ref)
    loc = self._page.raw.locator(selector).first
    
    # If targeting a label, find the associated input
    tag = loc.evaluate("el => el.tagName.toLowerCase()")
    if tag == "label":
        for_attr = loc.get_attribute("for")
        if for_attr:
            loc = self._page.raw.locator(f'input#{for_attr}').first
        else:
            # Try finding input inside the label
            loc = loc.locator('input[type="file"]').first
    
    loc.set_input_files(action.file)
```
